Frontend — Система учёта студентов (Student Accounting System)

Клиентская часть системы управления данными студентов факультета. Приложение позволяет просматривать списки студентов, детальные карточки с оценками, статистикой посещаемости и историей обучения.

---

Возможности

Список студентов — просмотр всех студентов с поиском и фильтрацией по группам и статусам  
Карточка студента — детальная информация, контакты, паспортные данные  
Успеваемость — оценки по предметам с цветовой индикацией  
Посещаемость — статистика и история посещений  
Интерфейс — современный дизайн в стиле dashboard с плавными анимациями  
Адаптивность — корректное отображение на планшетах и мобильных устройствах

API
Приложение работает с REST API. Базовый URL настраивается в src/api/index.js.

Метод	Endpoint Описание
GET	/students	Получить список всех студентов
GET	/students/:id	Получить данные конкретного студента
GET	/grades	Получить все оценки
GET	/attendance	Получить все записи посещаемости
GET	/subjects	Получить список предметов
Статус	Значение	Цвет
ACTIVE	Учится	Зелёный
ACADEMIC	Академический отпуск	Жёлтый
EXPELLED	Отчислен	Красный
GRADUATE	Выпускник	Серый
Оценка	Цвет	Описание
5	 Зелёный	Отлично
4	 Голубой	Хорошо
3	 Жёлтый	Удовлетворительно
2	 Красный	Неудовлетворительно
зачёт	 Зелёный	Зачёт сдан
незачёт	 Красный	Зачёт не сдан
